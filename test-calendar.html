Prueba de Calendario - Safari y Chrome

Instrucciones de Prueba:

  1. 1. Haz clic en el botón del calendario para abrirlo
  2. 2. Mueve el mouse sobre el input mientras el calendario está abierto
  3. 3. Verifica que el calendario NO se superponga con el input
  4. 4. Prueba en Safari y Chrome

✅ Prueba Exitosa Si:

  • • El calendario se abre correctamente
  • • Al pasar el mouse sobre el input, el calendario NO se superpone
  • • El calendario permanece visible y accesible
  • • Funciona igual en Safari y Chrome

❌ Prueba Fallida Si:

  • • El input se superpone al calendario al hacer hover
  • • El calendario desaparece o se oculta parcialmente
  • • Hay diferencias de comportamiento entre navegadores
